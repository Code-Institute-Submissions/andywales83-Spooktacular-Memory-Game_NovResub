Welcome to the Spooktacular Memory Game

The Spooktacular Memory Game is a card mathcing memory game, also known as Concentration. The nature of the game is a simple one, where players are faced with a deck of 12 cards all placed face down on a surface, and two cards are flipped over on each turn.


The object of the game, is for a player to turn over all cards and match the pairs.


This memory game, comes with the addition of a timer and start rating, allowing players the challenge of trying to match all cards in the fewest moves and in the quickest time.
